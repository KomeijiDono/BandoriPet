# ELECTRON-GSV-BandoriPet
基于ELECTRON架构开发的BanG Dream 桌宠程序。项目完全开源，可自行进行后续开发。
## Features
- Node 模块整合
- Electron API 调用
- 系统脚本调度
- 子进程管理
- IPC 通信
- Live2D 渲染
- AI 语音交互
- 桌面功能整合
## Requirements
- Windows 10/11
- Electron 20+
- Node.js 16+
- Python 3.10+
- C++ Runtime
- MSVC Runtime
- Visual Studio Build Tools
- GPT-SoVITS
